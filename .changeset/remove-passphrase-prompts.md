移除密码加密模式的所有密码输入提示

**改进**：
- 密码加密模式现在使用固定的内置密码，无需用户输入
- 移除所有密码相关的交互提示
- 创建配置时不再需要输入密码
- 启动时不再需要输入密码
- 加密模式迁移时不再需要输入密码

**技术细节**：
- 添加 `DEFAULT_PASSPHRASE` 常量作为固定密码
- 从 ConfigManager 移除 `sessionPassphrase` 和相关方法
- 从 UI 移除 `getPassphraseIfNeeded()` 方法
- 更新迁移工具，移除所有密码输入逻辑

**用户体验**：
- 密码加密模式完全透明，使用体验与 Keychain 模式一致
- 配置仍然是加密的，只是不需要用户记住密码
- 保持完全可移植性，可跨机器导入/导出
